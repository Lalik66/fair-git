## **Prompt: Route to Friend - Navigation Feature**

### Context:
I am building an interactive fair map using Mapbox GL JS and React. Users can see their friends' locations on the map. I need to add a "Get Directions" feature that shows a route from the user's current location to a selected friend's marker, similar to 2GIS navigation.

### Prerequisites:
- User location tracking is already implemented (Part 1 is complete)
- Friends' locations are displayed as markers on the map (Part 2 is complete)
- Mapbox GL JS map instance is available
- User has an active location (latitude/longitude)

---

### Task Requirements:

#### 1. UI/UX - Triggering Directions

**When user clicks on a friend marker:**
- Show a popup/card with friend's name and a "Get Directions" button
- Display estimated distance from user to friend (use Turf.js `turf.distance`)
- Button should be prominent and child-friendly (refer to ui-expert.md design system)

**Popup should include:**
- Friend's name and avatar/emoji
- Distance: "📍 250 meters away"
- "Get Directions 🧭" button (primary coral color)
- "Close" button to dismiss

---

#### 2. Mapbox Directions API Integration

**API Setup:**
- Use Mapbox Directions API: `https://api.mapbox.com/directions/v5/mapbox/{profile}/{coordinates}`
- Profile: Use `walking` (most appropriate for a fair)
- Include Mapbox access token from environment variables

**Request format:**
```
GET https://api.mapbox.com/directions/v5/mapbox/walking/{userLng},{userLat};{friendLng},{friendLat}
?geometries=geojson
&steps=true
&access_token={MAPBOX_TOKEN}
```

**Response handling:**
- Extract route geometry (GeoJSON LineString)
- Extract turn-by-turn steps
- Extract duration (in seconds) and distance (in meters)
- Handle errors (no route found, API error, friend too far away)

---

#### 3. Route Display on Map

**Route rendering:**
- Add a new layer to the Mapbox map with the route geometry
- Route line style:
  - Color: `#06BEE1` (sky-blue from design system)
  - Width: `5px` on mobile, `6px` on desktop
  - Line cap: `round`
  - Line join: `round`
  - Add subtle animation (optional): dashed line that appears to "move"

**Example route layer:**
```javascript
map.addSource('route', {
  type: 'geojson',
  data: {
    type: 'Feature',
    geometry: routeGeometry
  }
});

map.addLayer({
  id: 'route',
  type: 'line',
  source: 'route',
  layout: {
    'line-join': 'round',
    'line-cap': 'round'
  },
  paint: {
    'line-color': '#06BEE1',
    'line-width': 6,
    'line-opacity': 0.8
  }
});
```

**Start/End markers:**
- Show a marker at the user's location (current position) - different from friend marker
- Optionally pulse the friend's marker to indicate destination
- Auto-zoom/fit the map to show the entire route
